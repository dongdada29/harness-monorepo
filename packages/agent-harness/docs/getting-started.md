# 快速开始

## 环境要求

- Node.js 18+
- npm 9+

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

启动开发服务器，访问 http://localhost:3000

## 测试

```bash
npm test
```

## 构建

```bash
npm run build
```

## 项目结构

```
src/
├── types/      # 类型定义
├── config/     # 配置文件
├── services/   # 业务逻辑
├── ui/         # 界面组件
└── index.ts    # 入口
```
