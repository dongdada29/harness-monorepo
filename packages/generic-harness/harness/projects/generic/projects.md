# Generic Harness Project

> Platform: Generic  
> Version: 1.0.0

---

## 概述

通用项目 Harness 模板。

## 适用项目

- Node.js / TypeScript
- 任何框架
- 任何规模

## 命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 开发模式 |
| `npm run build` | 构建 |
| `npm run test` | 测试 |

## Gates

```
npm run lint → npm run typecheck → npm test → npm run build
```
