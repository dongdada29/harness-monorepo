## Checkpoint 详情

### CP0 — Init
- 理解任务
- 拆解目标
- 复杂度判断
- 人类确认（仅 High）

### CP1 — Plan
- 读 constraints.md
- 制定实现计划
- 确认接口设计

### CP2 — Exec
- 按计划实现
- 保持小步提交

### CP3 — Verify
- 运行 Gates
- 确保全部通过

### CP4 — Review
- 自审代码
- 清理调试代码
- 准备合并
